Пример абстрагирования:
Предположим, есть класс "Транспортное средство", который изначально был создан для общего представления различных
видов транспортных средств. Однако, по мере развития проекта стало ясно, что есть уникальные свойства и методы, к
оторые применимы только к моторным транспортным средствам.
Вместо того, чтобы добавлять специфические методы для моторных транспортных средств в исходный класс "Транспортное средство",
можно создать новый класс "Моторное транспортное средство" в качестве предка "Транспортного средства".

Пример факторизации:
Предположим, есть классы "Круг", "Прямоугольник" и "Треугольник", каждый из которых представляет геометрическую
фигуру с уникальными свойствами и методами. Однако, при дальнейшем анализе становится очевидным, что все эти классы
представляют собой частные случаи общего понятия "Геометрическая фигура".
Вместо того, чтобы дублировать общие свойства и методы в каждом классе, можно новый родительский класс
"Геометрическая фигура", который содержит общие для фигур методы
и может выступать интерфейсом.
Таким образом каждый класс "Круг", "Прямоугольник" и "Треугольник" становится подклассом этого родительского класса.
